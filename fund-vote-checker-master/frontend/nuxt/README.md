# Nuxt and Vuetify.js

## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# Optional: build for production and launch server
$ npm run build
$ npm run start

# We will most probably host on netlfy and generate a static project
$ npm run generate
```
